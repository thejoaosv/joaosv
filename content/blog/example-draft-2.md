This draft is not published because `draft` is set to `true`.

When you want to publish a post:

1. Copy this file.
2. Rename it to the URL slug you want, like `my-first-post.md`.
3. Change `draft` to `false`.
4. Write with normal Markdown.

## Markdown works

You can use **bold text**, _italic text_, links like [Trinnea](https://trinnea.com), lists, quotes, tables, and code blocks.

```tsx
export function TinyIdea() {
  return <p>Ship the small version first.</p>;
}
```

## Images work too

Put images in `public/blog/your-post/` and reference them with an absolute path:

```md
![A photo description](/blog/your-post/photo.png)
```
